## Step 1: Install the needed dependencies

use the following command

```bash


yarn install
```

## Step 2: Start your Application


### For Android

```bash

yarn android
```

### For iOS

```bash

yarn ios
```
## Step 3: Test the Application

```bash

yarn test
```
